# User GraphRAG Chat V1 Walkthrough

1. Start the backend, GraphRAG service, and frontend
2. Open the frontend
3. Navigate to `/smart-chat`
4. Ask: `Toi muon da toi nay, 10 nguoi, gia vua phai, gan Quan 7`
5. Confirm:
   - a natural-language answer appears
   - 2-3 recommendations are shown when candidate fields exist
   - each recommendation includes explicit reasons
6. If Neo4j or OpenRouter is unavailable, confirm the UI shows a safe fallback instead of unsupported claims
