## 🛠️ Công cụ & Cài đặt (Bắt buộc)

Để dự án chạy đồng nhất trên máy của mọi người, vui lòng cài đặt đầy đủ:

### 1. Phần mềm hỗ trợ
* **[Postman](https://www.postman.com/downloads/):** Công cụ dùng để kiểm thử (test) các luồng dữ liệu API.
* **[StarUML](https://staruml.io/):** Dùng để xem và chỉnh sửa các sơ đồ thiết kế (Class, Use Case, Sequence).

### 2. VS Code Extensions (Tiện ích mở rộng)
Mọi người vào mục Extensions trong VS Code và cài đúng các bản sau:
* ✨ **Prettier - Code formatter:** Tự động căn lề, xuống dòng cho đẹp code.
* 🛡️ **ESLint:** Cảnh báo các lỗi gõ code sai cú pháp ngay lập tức.
* 🌿 **GitLens:** Giúp theo dõi lịch sử chỉnh sửa code của từng người.
