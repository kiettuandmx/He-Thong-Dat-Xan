Chọn kiểu navigation cho AdminLayout

Cả ba phương án đều giữ visual hướng B: control center sáng, card nổi, nhấn xanh-teal. Khác nhau chủ yếu ở cách điều hướng.

1. Sidebar cố định bên trái

Điều hướng rõ ràng, chuyên nghiệp, phù hợp nếu admin làm việc lâu và chuyển nhiều màn hình trong ngày.

1 Admin
Điểm mạnh: ổn định, dễ scan, rất hợp dashboard và quản trị nhiều module.

2. Topbar + dropdown

Giữ gần cách dùng hiện tại, giao diện thoáng hơn nhưng điều hướng sâu sẽ chậm hơn khi số module tăng.

2 Admin
Điểm mạnh: ít thay đổi thói quen hiện tại, phù hợp nếu bạn muốn phần admin nhìn gọn và nhẹ hơn.

3. Hybrid

Desktop có sidebar để thao tác nhanh, mobile thu về topbar/drawer. Đây là hướng cân bằng nhất giữa trải nghiệm và responsive.

3 Admin
Điểm mạnh: desktop làm việc thoải mái như app quản trị thật, mobile vẫn gọn và dùng được.
